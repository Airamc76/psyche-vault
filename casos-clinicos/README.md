# Casos Clínicos — psyche-vault

Los casos clínicos son el componente más práctico del repositorio. Cada caso presenta
un paciente ficticio (pero clínicamente realista) y te guía a través del proceso de
formulación diagnóstica y planificación del tratamiento.

## Cómo usar los casos
1. Lee la presentación del caso sin ver las preguntas
2. Formulá tu hipótesis diagnóstica antes de continuar
3. Responde las preguntas de formulación
4. Lee el análisis del caso para comparar
5. Identifica qué perspectivas teóricas aporta cada elemento del caso
