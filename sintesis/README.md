# Sintesis — psyche-vault

Los modulos individuales dan profundidad. Este directorio da perspectiva: tablas comparativas, linea de tiempo y los debates que conectan todo.

## Contenido

- **escuelas-comparadas.md** — las 5 grandes tradiciones teoricas comparadas
- **terapias-comparadas.md** — TCC vs ACT vs DBT vs psicodinamica vs sistemica
- **linea-de-tiempo.md** — psicologia 1879-2024
- **debates-centrales.md** — 5 preguntas que la psicologia aun no resolvio
- **mapa-modulos.md** — como se conectan los 22 modulos entre si

## Para que sirve este directorio

Cuando terminas un modulo, lo entiendes en profundidad. Cuando lees sintesis, entiendes como ese modulo encaja en el conjunto. Son dos tipos de conocimiento distintos y complementarios.

Los archivos de este directorio no reemplazan los modulos: los contextualizan. Usa las tablas comparativas cuando necesites ubicar una escuela o terapia en relacion a las demas, no para aprender los detalles de cada una.

## Orden de lectura sugerido

1. Leer `mapa-modulos.md` para ver la arquitectura global del vault
2. Leer `linea-de-tiempo.md` para tener el eje historico
3. Leer `escuelas-comparadas.md` para comparar tradiciones teoricas
4. Leer `terapias-comparadas.md` para comparar aplicaciones clinicas
5. Leer `debates-centrales.md` para las preguntas abiertas que integran todo
