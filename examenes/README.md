# Exámenes — psyche-vault

El sistema de evaluación cubre los 22 módulos en cuatro parciales temáticos y un examen final integrador.

## Estructura

| Examen | Módulos | Formato |
|--------|---------|---------|
| [Parcial 1](parcial-01-bloque-A.md) | 01-04 (orígenes y escuelas) | 15 múltiple opción + 5 desarrollo + 2 ensayos |
| [Parcial 2](parcial-02-bloque-B.md) | 05-09 (neurociencias, social, desarrollo) | ídem |
| [Parcial 3](parcial-03-bloque-C.md) | 11-14 (metodología, procesos, cognición) | ídem |
| [Parcial 4](parcial-04-bloque-D.md) | 15-22 (ética, clínica, módulos aplicados) | ídem |
| [Final integrador](final-integrador.md) | 01-22 (todos) | 5 preguntas transversales + caso clínico + ensayo |

## Cómo usar los exámenes

1. Estudiá el módulo completo antes de hacer el examen parcial correspondiente
2. Respondé por escrito sin mirar las notas — el examen es un ejercicio de recuperación activa
3. Evaluá tu respuesta con la [rúbrica general](rubricas/rubrica-general.md) (5 dimensiones, 0-4 pts c/u)
4. El final integrador se hace cuando completaste todos los módulos

## Rúbrica

La [rúbrica general](rubricas/rubrica-general.md) evalúa cinco dimensiones en toda respuesta de desarrollo:
precisión conceptual, profundidad de análisis, evidencia empírica, integración de perspectivas, y posición propia razonada.
