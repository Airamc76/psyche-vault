# Módulo 00 — Orientación

## Qué es este módulo
El módulo cero no enseña psicología todavía. Prepara el terreno: por qué vale la pena
estudiar psicología de forma autodidacta, qué esperar, cómo funciona el aprendizaje
en esta disciplina, y cómo sacarle el jugo máximo al repo.

## Por qué existe
La psicología tiene un problema de entrada: todo el mundo cree que ya sabe algo
(porque hablamos de mente y conducta todo el tiempo), y eso crea falsa familiaridad.
Este módulo rompe eso. Te ayuda a entrar con la mente más limpia posible.

## Al terminar este módulo vas a poder
- Describir qué estudia la psicología y qué NO estudia
- Entender por qué existen tantas escuelas que no se ponen de acuerdo
- Tener un sistema propio para absorber y retener conocimiento psicológico
- Haber leído al menos una introducción general a la disciplina
- Tener un método propio de retrieval practice con `preguntas.md`, `respuestas.md`
  y Anki, sin depender de ninguna herramienta externa

## Estructura del módulo
- `lecturas.md` — Las primeras lecturas recomendadas
- `conceptos.md` — Qué es la psicología, cómo funciona el aprendizaje autodidacta
- `preguntas.md` — Las preguntas que estructuran la reflexión inicial
- `respuestas.md` — Las respuestas de referencia

## Cómo abordar este módulo
1. Leé este README completo
2. Leé `lecturas.md` y conseguí al menos el primer libro
3. Leé `conceptos.md` antes de arrancar con el libro
4. Mientras leés el libro, anotá en `notas/` lo que te genera preguntas
5. Respondé `preguntas.md` sin mirar las respuestas
6. Revisá `respuestas.md` honestamente

## Tiempo estimado
1-2 semanas, dependiendo de cuánto tiempo tengas por día.
